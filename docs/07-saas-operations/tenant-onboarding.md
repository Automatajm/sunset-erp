# TENANT ONBOARDING

## Self-Service Signup
1. User registers (email, password, company)
2. Auto-create tenant + default roles
3. 14-day free trial starts
4. Onboarding wizard (5 steps)
5. Email: Welcome + quick start guide

## Onboarding Wizard
- Company details
- Preferences (currency, timezone)
- Invite team members
- Import data (optional)
- Product tour

**Target:** 80% completion rate
