# PENETRATION TESTING

## Schedule
- Internal: Monthly (OWASP Top 10)
- External: Quarterly (full scope)
- Bug Bounty: After 6 months

## Scope
- Authentication bypass
- Authorization flaws
- SQL injection
- XSS vulnerabilities
- CSRF attacks
- Tenant isolation

## Remediation
- Critical: 7 days
- High: 30 days
- Medium: 60 days
- Low: 90 days
